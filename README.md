# Calculadora en Python

## Descripción

Calculadora de consola desarrollada en Python como práctica de programación básica.

Permite realizar operaciones matemáticas básicas entre dos números e incluye validaciones para evitar errores comunes.

## Funcionalidades

* Suma (+)
* Resta (-)
* Multiplicación (*)
* División (/)
* Validación de operaciones inválidas
* Manejo de errores cuando se ingresan valores no numéricos
* Prevención de división entre cero
* Opción para cerrar el programa escribiendo `salir`

## Tecnologías utilizadas

* Python

## Ejecución

1. Abrir una terminal.
2. Navegar hasta la carpeta del proyecto.
3. Ejecutar:

```bash
python calculadora.py
```

## Ejemplo de uso

```text
==== Calculadora ====
Escriba 'salir' para cerrar

Operación (+, -, *, /): +
Primer número: 10
Segundo número: 5
Resultado: 15
```
